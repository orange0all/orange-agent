## 三、技能管理

### 技能热加载

- **新增技能**：在 `skills/` 目录下新建 `.py` 或 `.md` 文件，**2秒内自动加载生效**
- **修改技能**：编辑已有技能文件，保存后自动更新
- **删除技能**：删除文件或文件夹，自动卸载
- **手动刷新**：输入 `reload`


### 技能类型

| 类型 | 文件格式 | 说明 |
|------|---------|------|
| Python 技能 | `.py` | 复杂逻辑（文件操作、系统调用等） |
| Markdown 技能 | `.md` | 纯对话型（翻译、总结、笑话等） |

### 技能计数规则

- 一个 `.py` 文件 = 一个技能
- 一个 `.md` 文件 = 一个技能
- 含 `SKILL.md` 的文件夹 = 一个技能（整体计为一个）
